DNS:

With content scattered all over the world, DNS or Doman Name System, allows internet users to reach sites hosted anywhere by using a simply standard naming criterion. Every networked computer has a unique IP Address which, like your home address or telephone number, has different parts which lead DNS servers to a specific location. Since IP Addresses are unique, complicated, and hard to remember, DNS allows the routing to be done via a string of letters which are associated with an IP address. This makes it so that users of skill level can navigate the internet with relative ease. There is no need to memorize long row of numbers like 64.233.191.255, when using a unique name like Google.com is much easier to remember.
