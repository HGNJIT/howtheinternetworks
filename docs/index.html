Domain names and ICANN

ICANN is an organization which is responsible for some of the most important protocols regarding IP Addresses and Domain Names.
IP Addresses, Packets and Routing

IP Addresses are unique identifiers which every networked computer is assigned. These addresses are used for identifying unique networks.
HTTP and HTTPS protocols

HTTP is the communication protocol used to pull information from remote web servers. Along with HTTPS, they make retreival and communication a breeze.
W3C and HTML and CSS

The W3C is the organization responsible for maintaining and developing standards and practices for web browsing.
DNS

DNS allows for the substitution of IP Addresses for Domain Names which makes finding and remembering websites exceptionally easy.
